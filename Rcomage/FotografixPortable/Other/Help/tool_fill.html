Using the Fill tool

The Fill tool is used to fill a region with a specific colour. It is equivalent to using the Magic Wand tool to select a region and then using the Fill command on the Edit menu to fill the selected region.

  1. Select the Fill tool by clicking on its icon in the tool box or by pressing F.
  2. Choose the foreground colour by using the colour palette or Dropper tool.
  3. Change the Magic Wand options, if required. See the Magic Wand tool for more information on these options.
  4. Click a point on the image to select the source point whose colour will be matched. Regions having colours similar to that of the source point will be filled with the foreground colour.

Note: If you are working with multiple layers, first choose the layer that you would like to fill.
