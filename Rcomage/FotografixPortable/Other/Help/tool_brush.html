Using the Brush tool

The Brush tool is used to paint on the canvas with a selected brush.

  1. Choose the foreground colour by using the colour palette or Dropper tool.
  2. Select the Brush tool by clicking on its icon in the tool box or by pressing B.
  3. In the options palette, choose the type of brush that you would like to paint with. You can also use the [ and ] keys to choose the previous and next brush respectively.
  4. Click and drag on the canvas to paint.

Note: If you are working with multiple layers, first choose the layer on which you want to paint.

Brush modifiers

Holding down certain keys while dragging modifies the brush behaviour.
