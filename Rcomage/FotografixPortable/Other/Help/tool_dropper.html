Using the Dropper tool

The Dropper tool is used to pick a colour from the image you are working on.

  1. Select the Dropper tool by clicking on its icon in the tool box or by pressing Y.
  2. Click the colour in the image that you want to pick as the foreground colour. You can also click and drag to find the colour you are looking for.

Note: If you are working with multiple layers, first choose the layer from which you want to pick the colour.
