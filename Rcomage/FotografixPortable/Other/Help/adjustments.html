Enhancing images

Fotografix provides a number of methods to digitally enhance photos and other images. These enhancements fall into two major categories:

  1. Basic adjustments such as brightness and contrast adjustment, available under the Adjustments menu
  2. Special effect filters such as motion blur, available under the Filter menu

All adjustments and filters apply only to the active layer. If your image contains multiple layers, first choose the layer that you would like to modify. Adjustments and filters can be further restricted to a portion of the image by selecting that portion first.

Using adjustment layers

An adjustment layer is a special type of layer that applies an adjustment to all layers below it. Unlike a regular adjustment, an adjustment layer does not modify the underlying image itself; instead the adjustment is dynamically applied to the underlying layers. In this way, you can try various adjustments without losing the original image.

To create an adjustment layer, choose one of the adjustments under the Layer > New Adjustment Layer menu. You can edit the adjustment at any time by using the Layer Properties command. For more information, see Working with layers.

If you select a portion of the image before creating an adjustment layer, the adjustment will be applied only to that portion. You can also include/exclude portions later by painting on the adjustment layer.
