What's New

Version 1.5 (1 June 2010)

  • New Gaussian Blur filter.
  • Multi-layer TIFF images and GIF animations can now be opened.
  • Middle mouse button now invokes the Hand tool for panning.
  • New blending modes: Darken, Lighten, Difference, Exclusion, Pin Light, Hard Mix.

Version 1.4.1 (10 April 2010)

  • Fixes for some minor localization issues.

Version 1.4 (8 April 2010)

  • Improved image resampling algorithm.
  • Support for multiple interface languages.

Version 1.3 (25 March 2010)

  • Improvements to status bar and other interface elements.
  • Additional shortcut keys for most menu commands. (most are Photoshop-compatible)
  • New Ruler tool for measuring distances and angles.
  • Simpler print units system; now includes centimetre as a unit.
  • Option to choose JPEG quality when saving JPEG images.

Version 1.2 (17 July 2009)

  • New Sharpen, Find Edges, Night Vision and Halftone filters.
  • Improved Diffuse and Emboss filters.
  • Improved scripting system.
  • Toolbox titles for easier identification.

Version 1.1 (27 June 2009)

Fotografix is now officially out of beta.

  • New Levels adjustment for adjusting shadows, midtones and highlights of an image.
  • Preset image sizes for creating new images: A4, letter, etc. and also dimensions of clipboard image, if available.
  • Mouse wheel can now be used for zooming.
  • New Pixelate filter.
  • New Rubber Stamp script.

Version 1.0 Beta 3 (19 June 2009)

  • Documentation for most features.
  • Edit > Paste External command for pasting images from the Windows clipboard. (the Paste command works only with the internal clipboard)
  • Layer > Merge Down and Layer > Flatten Image commands for merging layers.
