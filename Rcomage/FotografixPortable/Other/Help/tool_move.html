Using the Move tool

The Move tool is a versatile tool that can be used to do the following:

  • Move/copy a selected region of the image
  • Move the selection outline
  • Move an entire layer

To move/copy a selected portion of the image

  1. Select a region of the image using the Marquee tools or the Magic Wand tool.
  2. Select the Move tool by clicking on its icon in the tool box or by pressing V.
  3. Click and drag to move the selected region to a new position.
  4. To make a copy of the selected region instead of moving it, hold down Ctrl while dragging.

Note: If you are working with multiple layers, first choose the layer whose contents you would like to move/copy.

To move the selection outline

  1. Select a region of the image using the Marquee tools or the Magic Wand tool.
  2. Select the Move tool by clicking on its icon in the tool box or by pressing V.
  3. While holding down the Alt key, click and drag to move the selection outline to a new position.

To move an entire layer

  1. Choose the layer that you would like to move.
  2. Ensure that there is no existing selection. (if there is, use Select > Deselect from the menu or press Ctrl+D to remove the selection)
  3. Select the Move tool by clicking on its icon in the tool box or by pressing V.
  4. Click and drag to move the layer to its new position.
