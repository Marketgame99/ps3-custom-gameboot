Using the Clone tool

The Clone tool is used to clone a portion of the image at another location.

  1. Select the Clone tool by clicking on its icon in the tool box or by pressing C.
  2. Hold down the Alt key and click on the image to set the source point, i.e. the point from where you wish to clone.
  3. In the options palette, choose the type of brush that you would like to paint with.
  4. Click and drag on the canvas to clone from the source point.

Note: If you are working with multiple layers, first choose the layer on which you want to clone.

Brush modifiers

Holding down certain keys while dragging modifies the brush behaviour.
