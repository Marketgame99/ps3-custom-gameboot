Menu commands

File menu

CommandFunctionShortcut
NewCreates a new imageCtrl+N
OpenOpens an existing imageCtrl+O
Extract IconExtracts the display icon of a file
SaveSaves an imageCtrl+S
Save AsSaves an image with a new name or formatCtrl+Shift+S
CloseCloses the active imageCtrl+W
PrintPrints the active imageCtrl+P
Print PreviewPreviews the active image for printing
Page SetupChanges printing optionsCtrl+Shift+P
Choose LanguageChanges the interface languageCtrl+Q
ExitQuits the applicationCtrl+Q

Edit menu

CommandFunctionShortcut
UndoUndoes the last actionCtrl+Z / Ctrl+Alt+Z
RedoRedoes the previously undone actionShift+Z / Ctrl+Shift+Z / Ctrl+Y
CutRemoves the selected region and saves it on the internal clipboardCtrl+X
CopyCopies the selected region and saves it on the internal clipboardCtrl+C
PastePastes the contents of the internal clipboard as a new layerCtrl+V
Paste ExternalPastes the contents of the Windows clipboard as a new layerCtrl+Shift+V
ClearErases the selected region and makes it transparentDel
FillFills a layer or a selected region with a specified colourShift+F5
Purge (submenu)Contains commands to purge the contents of the undo buffer and/or the internal clipboard

Image menu

CommandFunctionShortcut
Image SizeResizes the pixel or print dimensions of the imageCtrl+Alt+I
Canvas SizeExtends or reduces the image canvasCtrl+Alt+C
CropCrops the image to the selected region
Reveal AllExtends the imaeg canvas to reveal any hidden layersCtrl+Shift+R
Flip (submenu)Contains commands to flip the canvas horizontally or vertically

Adjustments menu

CommandFunctionShortcut
Brightness/ContrastAdjusts brightness and contrast of the imageCtrl+Shift+B
Colour BalanceAdjusts colour balance of the imageCtrl+B
LevelsAdjusts shadows, midtones and highlights of the imageCtrl+L
DesaturateRemoves all colour from the imageCtrl+Shift+U
Black & WhiteConverts the image to black & whiteCtrl+Alt+Shift+B
Gradient MapApplies a gradient map using the current foreground and background colours
InvertInverts the colours in the imageCtrl+I
ThresholdApplies a threshold to the image
PosterizeReduces the number of colours used in the image

Note: To open an adjustment window with the preview option turned off, hold down the right Ctrl key on your keyboard while choosing the menu command.

Rasterize LayerRasterizes the active layer
New Adjustment Layer (submenu)Contains commands for creating new adjustment layers
Add Layer MaskAdds a layer mask to the active layer
