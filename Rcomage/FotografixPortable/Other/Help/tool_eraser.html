Using the Eraser tool

The Eraser tool is used to erase portions of the image with a selected brush.

  1. Select the Eraser tool by clicking on its icon in the tool box or by pressing Z.
  2. In the options palette, choose the type of brush that you would like to erase with.
  3. Click and drag on the canvas to erase. Erased portions are made transparent.

Note: If you are working with multiple layers, first choose the layer on which you want to erase.

Brush modifiers

Holding down certain keys while dragging modifies the brush behaviour.
