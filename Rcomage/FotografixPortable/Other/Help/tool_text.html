Using the Text tool

The Text tool is used to add text to an image.

  1. Select the Text tool by clicking on its icon in the tool box or by pressing T.
  2. Click on the image where you want to add text.
  3. The Text Properties window appears, allowing you to enter text and also choose the font and colour. After entering the text, click OK to add the text to the image.

Note: Text is always added as a new layer. You can modify the text at any time through the Layer Properties window. For more information, see Working with layers.
