Getting started

Used other image editors before?

If you have previously used one of the popular image editors such as Adobe Photoshop, GIMP or Paint.NET, then it should be quite easy for you to get started with Fotografix. The interface and much of the functionality of Fotografix is very similar to that of these software. For more information, see the following topics:

  • Interface overview
  • Quick Start Guide for Adobe Photoshop Users
