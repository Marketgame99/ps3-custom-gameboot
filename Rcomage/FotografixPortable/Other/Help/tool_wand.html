Using the Magic Wand tool

The Magic Wand tool is used to select regions of similar colour in an image.

  1. Select the Magic Wand tool by clicking on its icon in the tool box or by pressing W.
  2. The Tolerance setting in the options palette determines the range of colours that are selected. A value of 0 matches only the source colour and a value of 255 matches all colours.
  3. If the Contiguous setting is checked, only regions adjoining the source point are selected. If the setting is unchecked, all matching regions are selected.
  4. Click a point on the image to select the source point whose colour will be matched. Regions having colours similar to that of the source point will be selected.

Note: If you are working with multiple layers, first choose the layer on which the source point resides.

Selection modifiers

Holding down certain keys while clicking modifies the selection behaviour.

CtrlAdds the new selection to the existing selection
