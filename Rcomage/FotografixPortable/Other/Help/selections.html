Working with selections

Many operations in Fotografix can be restricted to a certain region of the image by using a selection. You can select a region using the Marquee tools or the Magic Wand tool. When you select a portion of the image, a selection outline appears along the boundary of the selection. You can further modify the selection outline using the commands under the Select menu.

The following operations in Fotografix can work with selected regions of an image:

  • The Cut, Copy, Clear and Fill commands under the Edit menu
  • The Crop command under the Image menu
  • All adjustments and most filters
  • The Move tool
