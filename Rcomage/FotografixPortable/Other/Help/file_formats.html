Supported file formats

Fotografix supports opening and saving images in a variety of image file formats. The following table lists the supported formats and features of each format.

FormatFile extension(s)Colour depthLayersWrite
Fotografixfgx32 bppYesYes
Photoshoppsd32 bppYesNo
GIMPxcf32 bppYesNo
JPEGjpg, jpeg, jpe, jfif24 bppNoYes
Bitmapbmp24 bppNoYes
PNGpng32 bppNoYes
GIFgif8 bppYes *Yes
TIFFtif, tiff32 bppYes (read-only)Yes
Targatga, vda, icb, vst32 bppNoYes
PCXpcx8 bppNoNo
Windows Iconico, cur32 bppNoNo
RAWraw24 bppNoYes

Notes

  • Colour depth: bpp stands for bits per pixel, which determines the maximum number of colours that the format can store.
    • 8 bpp - 256 colours
    • 24 bpp - 16 million colours
    • 32 bpp - 16 million colours + transparency
  • Layers: Specifies whether the format supports multiple layers.
  • Write: Specifies whether Fotografix is capable of saving files in this format. (All listed formats can be opened)

* Individual frames in an animated GIF file are imported as layers. The reverse (saving an animated GIF) is currently not possible.
