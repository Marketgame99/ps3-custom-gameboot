Using the colour palette

The colour palette displays the current foreground and background colours. To pick a new colour, click the corresponding square in the colour palette and choose a new colour using the colour picker.

  • To swap the foreground colour and the background colour, click the two-headed arrow in the colour palette. Alternatively, you can press X on the keyboard.
  • To reset the colours to the default black and white, click the small black-and-white squares in the colour palette. Alternatively, you can press D on the keyboard.

Further, there are two keyboard shortcuts related to filling using the colour palette:

  • Alt+Del fills the image (or the selected region) with the foreground colour.
  • Ctrl+Del fills the image (or the selected region) with the background colour.
